数组的创建

数组方法

push: 向数组末尾添加元素

pop: 删除数组末尾的元素

shift: 删除数组开头的元素

unshift: 向数组开头添加元素

splice: 在数组中添加或删除元素

sort: 对数组元素进行排序

reverse: 反转数组元素的顺序

join: 将数组元素转换为字符串


var arr = [1, 3, 2];
arr.push(4); // [1, 3, 2, 4]
arr.pop(); // 4
arr.shift(); // 1
arr.unshift(0); // [0, 3, 2]
arr.splice(1, 1, "apple"); // [0, "apple", 2]
arr.sort(); // [0, 2, "apple"]
arr.reverse(); // ["apple", 2, 0]
arr.join("-"); // "apple-2-0"

遍历数组

for...in: 遍历数组的键值

forEach: 遍历数组的元素


var arr = [1, 2, 3];
// for...in
for (var i in arr) { console.log(arr[i]); }
// forEach
arr.forEach(function(e) { console.log(e); });
// 箭头函数
arr.forEach((e) => { console.log(e); });

字符串方法

charAt: 获取指定位置的字符

charCodeAt: 获取指定位置的字符的ASCII码

concat: 连接两个或多个字符串

indexOf: 查找指定字符串的位置

lastIndexOf: 从后向前查找指定字符串的位置

localeCompare: 比较两个字符串

match: 匹配字符串

replace: 替换字符串

search: 查找字符串

slice: 截取字符串

split: 分割字符串

substring: 截取字符串

toLowerCase: 转换为小写

toUpperCase: 转换为大写

trim: 去除字符串两端的空白字符


var str = "Hello, world!";
console.log(str.charAt(0)); // "H"
console.log(str.charCodeAt(0)); // 72
console.log(str.concat(" How are you?")); // "Hello, world! How are you?"
console.log(str.indexOf("l")); // 2
console.log(str.lastIndexOf("l")); // 9
console.log(str.localeCompare("hello")); // 1
console.log(str.match(/l/g)); // ["l", "l"]
console.log(str.replace("l", "L")); // "HeLLo, world!"
console.log(str.search("l")); // 2
console.log(str.slice(2, 5)); // "llo"
console.log(str.split(" ")); // ["Hello,", "world!"]
console.log(str.substring(2, 5)); // "llo"
console.log(str.toLowerCase()); // "hello, world!"
console.log(str.toUpperCase()); // "HELLO, WORLD!"
console.log(str.trim()); // "Hello, world!"

JSON

JSON(JavaScript Object Notation)是一种轻量级的数据交换格式，它基于ECMAScript的一个子集。它是一种基于文本的、可读性高的、占用空间小的的数据交换格式。

JSON的创建


var obj = {name: "John", age: 30, city: "New York"};
var json = JSON.stringify(obj);
console.log(json); // {"name":"John","age":30,"city":"New York"}

JSON的解析


var json = '{"name":"John","age":30,"city":"New York"}';
var obj = JSON.parse(json);
console.log(obj.name); // "John"

JSON.stringify()方法用于将JavaScript对象转换为JSON字符串。

JSON.parse()方法用于将JSON字符串转换为JavaScript对象。


                        var obj = {name: "John", age: 30, city: "New York"};
                        var json = JSON.stringify(obj);
                        console.log(json); // {"name":"John","age":30,"city":"New York"}
                        var obj2 = JSON.parse(json);
                        console.log(obj2.name); // "John"

对象：用花括号括起来的键-值对列表，键和值之间用冒号分隔

null：表示一个空值

Vue

Vue组件

Vue组件是Vue中重要的概念，它允许我们定义可重用的HTML、CSS和JavaScript代码片段，并在不同的地方重复使用。

创建组件

Vue组件是一个带有预定义选项的Vue实例，它包含了数据、模板、方法、生命周期钩子等。

// 定义一个名为 button-counter 的组件
Vue.component('button-counter', {
    template: '<button v-on:click="increment">{{ count }}</button>',
    data: function() {
        return {
            count: 0
        }
    },
    methods: {
        increment: function() {
            this.count++;
        }
    }
});

// 使用组件
<div id="app">
    <button-counter></button-counter>
</div>
<script>
    var app = new Vue({
        el: '#app'
    })
</script>

组件的事件

组件可以触发自定义事件，并通过事件回调来响应。

// 定义一个名为 button-counter 的组件
Vue.component('button-counter', {
    template: '<button v-on:click="increment">{{ count }}</button>',
    data: function() {
        return {
            count: 0
        }
    },
    methods: {
        increment: function() {
            this.count++;
            this.$emit('increment');
        }
    }
});

// 使用组件
<div id="app">
    <button-counter v-on:increment="incrementCounter"></button-counter>
    <p>{{ counter }}</p>
</div>
<script>
    var app = new Vue({
        el: '#app',
        data: {
            counter: 0
        },
        methods: {
            incrementCounter: function() {
                this.counter++;
            }
        }
    })
</script>
效果：

{{ counter }}
